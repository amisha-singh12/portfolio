## 🚀 Run Locally  

### Clone the project  
```bash
git clone https://github.com/amisha-singh12/MyTechfolio.git
```

### Go to the project directory  
```bash
cd MyTechfolio
```

### Install dependencies
```bash
npm install
```
### Start the development server
```bash
npm run dev
```
